### Simple Quiz game 
- Project buil in order to practice applying Typescript with React

### Prequisites:
- Node v12
- Typescript v3

### Launching project on localhost:3000 :
- npm install
- npm start 

### Technologies used:
- Typescript
- React 
- Styled Components

### Deployment: 
- Netlify [deployed version]('https://beach-react-ts-quiz.netlify.app')
